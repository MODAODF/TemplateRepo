## TemplateRepo

TemplateRepo 提供機關連結自建的 ODF 範本中心服務，建立集中管理並統一更新的範本服務

## Linux產生TemplateRepo.oxt
	1. 在clone or download 選擇 Download ZIP
	2. 解壓縮檔zip
	3. 進入解壓目錄
	4. 執行make

## Windows產生TemplateRepo.oxt
	1. 在clone or download 選擇 Download ZIP
	2. 選擇檔案按右鍵~選Extract All~
	3. 進入解壓目錄
	4. 選擇runoxt.bat~doubleclick 或按 Enter 執行
